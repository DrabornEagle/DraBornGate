# DraBornGate Supabase

DraBornGate, mevcut DraBornGo Supabase projesini kullanır.

- Kimlik ortak: `auth.users`
- Ürün verisi ayrı: `draborngate.dkd_gate_*`
- Mobil erişim kapıları: `public.dkd_gate_*`
- Demo kayıtları kullanıcıya özel ve sürümlüdür.

Canlı projede uygulanan migrationlar:

- `draborngate_v0_0_2_core`
- `draborngate_v0_0_2_security_hardening`
- `draborngate_v0_0_2_performance_hardening`
- `draborngate_v0_0_2_bootstrap_sites`
